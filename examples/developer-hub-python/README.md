# developer-hub-python

## Tooling (recommended)

- [pyenv](https://github.com/pyenv/pyenv)
- [poetry](https://python-poetry.org)

## Install dependencies

```bash
poetry install
```

## Format

```bash
poetry run ruff format
```

## Lint

```bash
poetry run ruff check
```

## Test

```bash
poetry run pytest
```
